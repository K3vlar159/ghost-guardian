# Night Terrors: Ghost Guardian

Projekt hry **Night Terrors: Ghost Guardian**, ktorú sme vytvorili počas **48h game jamu** v tíme 4 ľudí. Kde moja rola bola main coding (movement, enemy logic, stavba levelu, UI, boss encounter)

Hra je spravená v **Godot Engine** (môj prvý kontakt s Godotom – veľa vecí vznikalo „za jazdy“).
